# Python Hello World

Learning Python basics.

## Files
- 0-run: runs python file
- 1-run_inline: runs python code inline
- 2-print.py: prints a string
- 3-print_number.py: prints number with text
- 4-print_float.py: prints float with 2 decimals

- 6-concat.py: string concatenation
- 7-edges.py: string slicing practice
- 8-concat_edges.py: extracting parts of strings
- 9-easter_egg.py: the zen of python

